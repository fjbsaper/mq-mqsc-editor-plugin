This is an overview of the MQSC Scripts plugin for MQExplorer and Eclipse.

This Eclipse plugin provides all of the GUI tools for managing, editing and executing MQSC Scripts in the MQExplorer workspace.

It includes features to import, create, delete and otherwise handle MQSC files, Eclipse projects and folders underneath Eclipse projects.

This Eclipse project contains code to perform all Actions, manage eclipse preferences, display dialogs, and update the MQ Explorer Navigator tree to show all relevant Eclipse objects 99909that contain MQSC scripts - files that end with "".mqsc" and several other common file extensions for MQSC files.
